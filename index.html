Esta pagina web fue creada con el fin de ayudar a las personas que nesecitan informacion sobre los temas aqui presentados con la intencion de minimizar un poco su busqueda y que le sea util para realizar su trabajo, ya ue encontrara informacion clara, precisa y resumida con textos cortos e imagenes sobre su funcionamiento.



PROPOSITO DEL SITIO WEB


El proposito principal de este sitio web es dar a conocer información sobre el diseño de sitios web, aqui se selecciona informacion clasificada de manera resumida para que los usuarios conozcan un poco mas sobre las etiquetas semanticas y sobre CSS Grid.




INTRODUCCION
